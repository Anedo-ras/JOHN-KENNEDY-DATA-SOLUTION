John Kennedy Data Solutions - Price Card UI
A modern, responsive HTML/CSS price card interface for "John Kennedy Data Solutions" showcasing data, minute, and SMS bundles with an improved visual design.

📋 Overview
This project transforms a basic text-based price list into an elegant, user-friendly card layout. It maintains all original offer details while presenting them in a clean, professional interface suitable for mobile or desktop display.

✨ Features
Modern Aesthetic: Clean card design with gradient background, subtle shadows, and hover effects

Responsive Layout: Adapts seamlessly from mobile to desktop screens

Organized Categories: Separate sections for Data, Minutes, and SMS offers

Branded Header: Features updated business name and helpline (0745871285)

Interactive Elements: Hover effects on offer cards and call button

Preserved Data: All original offers (including duplicate minute entries) are maintained

🎨 Design Highlights
Color Scheme: Deep ocean blue (#0b2a3e) with warm amber accents (#f8b042)

Typography: Clean Inter font family with proper hierarchy

Spacing: Generous padding and gaps for readability

Visual Cues: Icons for each category, price highlighting, subtle duplicate tags

📱 Sections
Header: Business name, TILL number, and updated helpline

Data Offers: 11 different data bundle options

Minutes Offers: 4 minute bundle options (with duplicate differentiation)

SMS Offers: 5 SMS bundle options

Footer: Call-to-action with clickable helpline

🚀 Usage
Simply open the HTML file in any modern web browser. The design is self-contained with inline CSS and Font Awesome icons via CDN.

Colors: Modify the CSS variables in the brand-header, offer-price, and cat-title classes

Offers: Update the offer items in each grid section while maintaining the same structure

Contact Info: Change the TILL number and helpline in the header and footer sections
